# Rss-Translation

这是一个外文RSS翻译转换订阅页面 

[![circle_translate](https://github.com/ototooo/Rss-Translation/actions/workflows/circle_translate.yml/badge.svg)](https://github.com/tjsky/Rss-Translation/actions/workflows/circle_translate.yml)
[![Deploy](https://github.com/ototooo/Rss-Translation/actions/workflows/jekyll-gh-pages.yml/badge.svg)](https://github.com/tjsky/Rss-Translation/actions/workflows/jekyll-gh-pages.yml)

在 [test.ini](https://github.com/ototooo/Rss-Translation/blob/main/test.ini) 中提交以转换订阅，转换[rss to json](https://rss2json.com/)

查看[ RSS-Translation ](https://tjsky.github.io/RSS-Translation)使用[ 教程 ](https://www.tjsky.net/tutorial/644)修改自[ Rss-translate ](https://github.com/rcy1314/Rss-Translation/)
